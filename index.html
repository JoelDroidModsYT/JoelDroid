🌐 Sincronización Global (GitHub)

Conecta con un GitHub Gist para que los cambios del panel apliquen a TODOS los visitantes en tiempo real.

Pasos: 1) Ve a github.com → Settings → Developer settings → Personal access tokens → New token (selecciona scope "gist"). 2) Crea un Gist público vacío en gist.github.com. 3) Pega el token y el ID del Gist aquí.

📦 JSONBin.io (Recomendado — Gratis)

1) Ve a jsonbin.io → Regístrate gratis
2) Bins → Create Bin → pega {} → Save
3) Copia el BIN ID y tu Master Key
4) Pégalos en SYNC_CONFIG al inicio del HTML
